# Medicine

Лабораторная работа по проектированию баз данных в с использованием веб-фреймворка Django

## Развертывание проекта на локальной машине (Windows)

1. Скопируйте репозиторий на локальный компьютер (необходимо наличие установленного на компьюетере гита): ```git clone https://github.com/Maglctea/medicine.git```
2. Скачайте и установите Python (при необходимости) *https://www.python.org/*
3. Создайте виртуальное окружение в корне проекта `python -m venv env`
4. Активируйте виртуальное окружение `.\env\Scripts\activate`
5. Загрузите зависимости `pip install -r requirements.txt`
6. Выполните миграции в базу данных `python manage.py migrate`
7. Запустите приложение `python manage.py runserver` (по умолчанию запускается 127.0.0.1:8000)

## Использование

* Вход в панель управления (логин и пароль **admin**): http://127.0.0.1:8000/
* Отчет по продажам медикаментов http://127.0.0.1:8000/apteka/report/
